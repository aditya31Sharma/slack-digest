Setup required
1

Create a Slack App

Go to api.slack.com/apps → Create New App → From scratch.
Give it a name (e.g. "Summarizer") and select your workspace.

2

Add OAuth Scopes

Under OAuth & Permissions → Bot Token Scopes, add these scopes:

channels:history channels:read groups:history groups:read im:history im:read im:write mpim:history mpim:read users:read chat:write
3

Install & get Bot Token

Click Install to Workspace. Copy the Bot User OAuth Token (starts with xoxb-).

4

Find your Slack User ID

In Slack, click your name → Profile → ⋯ More → Copy member ID. It looks like U0XXXXXXXXX.

5

Get Anthropic API Key

Sign up at console.anthropic.com, go to API Keys and create a new key.

6

Create .env file

In the project folder (slack-summarizer/), create a .env file:

SLACK_BOT_TOKEN=xoxb-your-token-here
SLACK_USER_ID=U0XXXXXXXXX
ANTHROPIC_API_KEY=sk-ant-your-key-here

Then restart the server: npm start

7

Invite bot to channels

In each Slack channel you want summarized, type /invite @YourBotName.
